# Unraid Templates

Docker templates for Unraid Community Applications.

## Available Templates

### Facet

A self-hosted personal profile platform. Create multiple views of your professional profile for different audiences (recruiters, conferences, clients). Own your data, skip the tracking.

- **Project**: https://github.com/jesposito/Facet
- **Docker Image**: `ghcr.io/jesposito/facet:latest`
- **Support**: https://github.com/jesposito/Facet/issues

**Features:**
- Multiple profile "views" with different content for different audiences
- Privacy controls (public, unlisted, password-protected, private)
- GitHub import with optional AI summaries
- Resume upload and AI parsing (PDF/DOCX)
- AI writing assistant for content improvement
- RSS feed and iCal export for talks
- Single Docker container, SQLite database

## Installation

These templates are available through Unraid Community Applications. Search for the app name in the Apps tab.
